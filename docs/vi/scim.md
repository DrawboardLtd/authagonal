# Cung cấp SCIM 2.0

Authagonal hỗ trợ SCIM 2.0 để cung cấp người dùng tự động thông qua các nhà cung cấp danh tính doanh nghiệp như Microsoft Entra ID, Okta và OneLogin.

Để biết chi tiết cấu hình và các endpoint, vui lòng tham khảo [tài liệu tiếng Anh](/docs/scim).
