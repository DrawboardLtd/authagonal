# Provisionamento SCIM 2.0

Authagonal oferece suporte ao SCIM 2.0 para o provisionamento automático de usuários por meio de provedores de identidade empresariais como Microsoft Entra ID, Okta e OneLogin.

Para detalhes de configuração e endpoints, consulte a [documentação em inglês](/docs/scim).
