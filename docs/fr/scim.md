# Provisionnement SCIM 2.0

Authagonal prend en charge SCIM 2.0 pour le provisionnement automatique des utilisateurs via des fournisseurs d'identité d'entreprise tels que Microsoft Entra ID, Okta et OneLogin.

Pour les détails de configuration et les endpoints, consultez la [documentation en anglais](/docs/scim).
