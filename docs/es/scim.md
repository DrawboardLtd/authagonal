# Aprovisionamiento SCIM 2.0

Authagonal es compatible con SCIM 2.0 para el aprovisionamiento automático de usuarios a través de proveedores de identidad empresariales como Microsoft Entra ID, Okta y OneLogin.

Para obtener detalles de configuración y endpoints, consulte la [documentación en inglés](/docs/scim).
