# SCIM 2.0-Bereitstellung

Authagonal unterstützt SCIM 2.0 für die automatische Benutzerbereitstellung durch Enterprise-Identitätsanbieter wie Microsoft Entra ID, Okta und OneLogin.

Konfigurationsdetails und Endpunkte finden Sie in der [englischen Dokumentation](/docs/scim).
